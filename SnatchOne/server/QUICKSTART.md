# 🚀 Quick Start Guide - Snatch Unified Server

## Быстрый запуск за 3 шага

### Шаг 1: Установка зависимостей
```bash
cd server
npm install
```

### Шаг 2: Запуск сервера
```bash
npm start
```

### Шаг 3: Откройте админ-панель
Откройте в браузере: **http://localhost:3000/admin**

**Логин**: `admin`  
**Пароль**: `admin123`

---

## 📋 Что дальше?

### 1. Создайте первую лицензию
1. Войдите в админ-панель
2. Нажмите "Генерация ключей"
3. Укажите количество дней и описание
4. Скопируйте сгенерированный ключ

### 2. Настройте расширение Chrome
1. Откройте расширение Snatch
2. В настройках укажите URL сервера: `http://localhost:3000`
3. Вставьте лицензионный ключ
4. Нажмите "Start Bot"

### 3. Мониторинг
- Следите за активными сессиями в админ-панели
- Проверяйте статистику отправленных сообщений
- Управляйте операторами

---

## 🧪 Тестирование производительности

Для проверки работы с 100+ пользователями:

```bash
# Сначала создайте тестовый ключ в админке
# Затем запустите тест:
TEST_AUTH_KEY=ваш_ключ npm run test:load
```

Параметры теста:
```bash
# Количество подключений (по умолчанию 100)
NUM_CONNECTIONS=150 TEST_AUTH_KEY=ваш_ключ npm run test:load

# Длительность теста в миллисекундах (по умолчанию 60000)
TEST_DURATION=120000 TEST_AUTH_KEY=ваш_ключ npm run test:load
```

## 🔧 Настройка для продакшена

### 1. Создайте файл `.env`:
```bash
cp .env.example .env
```

### 2. Отредактируйте `.env`:
```env
PORT=3000
HOST=0.0.0.0
JWT_SECRET=ваш_секретный_ключ_измените_это
PUBLIC_URL=https://ваш-домен.com
```

### 3. Запустите с PM2 (рекомендуется):
```bash
npm install -g pm2
pm2 start server.js --name snatch-server
pm2 save
pm2 startup
```

## ❓ Частые вопросы

### Порт 3000 уже занят?
Измените порт в `.env` файле:
```env
PORT=4000
```

### Не могу войти в админку?
Проверьте, что используете правильные данные:
- Логин: `admin`
- Пароль: `admin123`

Если забыли пароль, пересоздайте базу данных или используйте SQL для сброса.

### Расширение не подключается?
1. Проверьте, что сервер запущен
2. Убедитесь, что URL в расширении правильный: `http://localhost:3000`
3. Проверьте, что лицензионный ключ действителен
4. Посмотрите логи сервера на наличие ошибок

### Как добавить нового администратора?
1. Войдите как SuperAdmin
2. Перейдите в раздел "Пользователи"
3. Нажмите "Добавить пользователя"
4. Выберите роль "admin"

## 🆘 Поддержка

Если возникли проблемы:

1. **Проверьте логи** - они содержат подробную информацию об ошибках
2. **Перезапустите сервер** - иногда это решает проблему
3. **Проверьте базу данных** - убедитесь, что файл `database.sqlite` существует
4. **Проверьте версию Node.js** - требуется версия 16 или выше
